Implementación de Servicios RESTful

En esta actividad, se desarrollará una aplicación que implementa servicios RESTful utilizando una base de datos. La aplicación permitirá gestionar Alumnos y Cursos, así como la inscripción de los alumnos en los diferentes cursos disponibles. Por medio de la aplicación cliente, se hará uso de los métodos GET, POST, PUT, y DELETE para crear, modificar, visualizar y eliminar tanto alumnos como cursos, y gestionar las inscripciones de manera eficiente.
